# 财务字段与可选映射设计

## Summary

本轮在现有 `订单总表 + 录单页 + 历史工作台` 的基础上，新增一套面向利润核算的轻量财务扩展能力：

- 映射字段支持停用，不想写飞书的字段可以直接留空并在界面中隐藏
- 录单页新增固定财务字段
- 录单页新增 3 个可命名的自定义成本字段
- 系统自动计算平台扣点金额、采购总成本和毛利润
- 历史页完整保存并可编辑这些财务字段

目标是让用户在录单时顺手补齐财务信息，而不是把系统做成无限制的表单搭建器。

## Goals

- 允许用户不使用不需要的字段映射
- 支持录单时填写平台扣点等财务信息
- 支持 3 个自定义成本字段，名称可配置
- 自动计算实际毛利润
- 历史记录和飞书写入都能承接这些字段

## Non-Goals

- 不做无限数量的自定义字段
- 不做复杂公式编辑器
- 不做按店铺分别定义不同财务字段
- 不做飞书列的自动创建

## Design

### 1. 映射字段改为可停用

设置页中的总表字段映射继续保留全部内置字段，但增加两个交互规则：

- 映射输入框留空时，表示该字段停用，不写入飞书
- 可通过“仅显示启用字段”切换，默认显示全部，避免初次使用时迷失

保留现有逻辑：

- 目标列名为空时，不进入飞书 payload
- 检测总表字段时，只检查当前已启用的目标列

这样用户不想要的字段不需要真正删除模型，只需停用映射即可。

### 2. 固定财务字段

在订单录入页新增以下固定财务字段：

- 平台扣点比例
- 平台扣点金额
- 其他成本
- 采购总成本
- 毛利润

其中：

- `平台扣点比例` 允许手填百分比或普通数字
- `平台扣点金额` 默认自动计算，也允许手动覆盖
- `采购总成本` 自动由 3 组采购位汇总
- `毛利润` 自动计算，默认只读展示

默认公式：

`毛利润 = 收入 - 平台扣点金额 - 采购总成本 - 其他成本 - 自定义成本合计`

### 3. 三个自定义成本字段

新增 3 个全局自定义成本槽位，每个槽位包含：

- 字段名称
- 字段值

设置页中允许配置默认名称，例如：

- 包装费
- 赠品成本
- 补贴

录单页里展示为：

- 自定义字段 1 名称对应的输入框
- 自定义字段 2 名称对应的输入框
- 自定义字段 3 名称对应的输入框

若名称为空，则该字段在录单页隐藏，不参与计算。

### 4. 录单页交互

录单页的财务区块位于采购信息下方，分为两段：

- 固定财务字段
- 自定义成本字段

交互规则：

- 收入变化时，自动重算扣点金额和毛利润
- 平台扣点比例变化时，自动重算扣点金额和毛利润
- 采购位中的数量或成本变化时，自动重算采购总成本和毛利润
- 自定义成本变化时，自动重算毛利润

若用户手动修改 `平台扣点金额`，则本次订单以内手动值优先，直到比例再次被修改。

### 5. 数据模型

订单模型新增以下财务字段：

- platform_fee_rate
- platform_fee_amount
- other_cost
- procurement_total_cost
- gross_profit
- custom_cost_labels
- custom_cost_values

其中：

- `custom_cost_labels` 固定为长度 3 的名称列表
- `custom_cost_values` 固定为长度 3 的数值列表

采购位仍保留现有 3 组：

- 商品
- 数量
- 成本

### 6. 飞书映射

在总表字段映射中新增这些可选字段：

- 平台扣点比例
- 平台扣点金额
- 其他成本
- 采购总成本
- 毛利润
- 自定义字段1
- 自定义字段2
- 自定义字段3

说明：

- 自定义字段只写值，不写标签
- 如果用户希望在飞书里保留字段含义，应将飞书列名直接命名为业务名称，如“包装费”
- 程序端映射名称保持固定，便于兼容历史记录和配置

### 7. 历史页

历史页详情默认展示并可直接编辑：

- 平台
- 固定财务字段
- 3 个自定义成本字段

保存修改后：

- 历史快照更新
- 毛利润相关字段重新计算并落盘
- 重新写入飞书时使用最新值

### 8. 配置存储

设置页新增全局财务配置：

- custom_cost_label_1
- custom_cost_label_2
- custom_cost_label_3
- show_only_enabled_mappings

历史配置兼容规则：

- 未配置自定义字段名称时，默认隐藏自定义字段输入框
- 未配置新财务映射时，视为停用
- 老历史记录缺失新财务字段时，按空值加载并自动补默认结构

## Error Handling

- 数字字段允许为空，空值按 `0` 参与计算
- 非法数字输入不阻止录单，但该字段边框高亮并在状态栏提示
- 飞书映射为空时仅视为停用，不视为错误
- 飞书字段检测只针对启用映射

## Testing

- 设置页可停用映射并正确保存/加载
- 设置页可配置 3 个自定义字段名称
- 录单页财务字段输入会实时更新平台扣点金额、采购总成本、毛利润
- 采购 1/2/3 变化能正确汇总采购总成本
- 自定义字段名称为空时，对应输入框隐藏
- 历史页可展示并编辑财务字段
- 历史页重新写入飞书会带上启用的财务字段
- 老配置与老历史数据仍可正常加载

## Recommendation

先按“固定财务字段 + 3 个自定义成本字段”实现，不开放更多自由字段。这样能最快支持你现在的利润核算，同时保持飞书映射、历史兼容和页面布局可控。
